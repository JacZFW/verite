# Verity

## Packages

- [demo-site](./packages/demo-site)

## Getting Started

```
npm run bootstrap
```

## Running the apps

### demo-site:

```
npm run demo-site
```

### Developing

To run type-checking, linting, and tests, simply run:

```
npm run check
```

### Linting the codebase

```
npm run lint
```

### Fixing with Prettier

This app uses [Prettier](https://prettier.io), and you can auto-format all files with

```
npm run format
```
